###Run/Develop:

#### Requirements:
- bower, gulp, sass (ruby)

```
npm install -g bower gulp
gem install sass
```

#### Setup
1. Clone Repo
2. Install node modules and bower components / dependencies
```
npm install && bower install
```
3. Run
```
gulp serve
```
